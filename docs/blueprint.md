# **App Name**: RationQ

## Core Features:

- Ration Card Validation: Verifies the ration card number entered by the user.
- Slot Availability Display: Shows available time slots based on pre-booking and shop timings.
- Inventory Display: Displays a list of available inventory items for the selected shop.
- Slot Selection and Booking: Allows users to select a time slot and book it.
- Notifications: Sends booking confirmation and reminders via email and SMS.
- OTP Generation and Validation: Generates an OTP upon booking and validates it when the customer arrives.
- Dynamic Stock Management: Uses a tool to monitor live inventory updates triggered either by shopkeeper restock activities, or from the recording of completed sales, providing accurate inventory information in real time.
- Sales Recording UI: Allows shopkeepers to input items and generate bills after OTP validation.
- Sales Recording Backend: Records sales in a SQL database.
- Admin Shop Management: The admin can add or remove multiple shops.
- Admin User Management: Allows the admin to manage user ration card numbers and assignments to shops.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for reliability and trust.
- Background color: Light blue (#E3F2FD), desaturated, to create a light, calm atmosphere.
- Accent color: Complementary orange (#FFB74D) to highlight important actions and information.
- Body and headline font: 'PT Sans', sans-serif, offering a modern yet approachable feel, ensuring readability for all users.
- Use simple, easily recognizable icons to represent inventory items and actions.
- A clean, straightforward layout with clear information hierarchy to ensure ease of use for all users, including those less familiar with technology.
- Use subtle transitions and loading animations to enhance user experience without overwhelming them.
